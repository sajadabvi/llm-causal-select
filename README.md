# llm-causal-select

**LoRA fine-tuned LLM for causal DAG selection from RASL equivalence classes.**

Extends the [gunfolds](https://github.com/sajadabvi/gunfolds) causal discovery toolkit by replacing manual or heuristic DAG selection with a learned LLM-based selector.

---

## Motivation

The gunfolds RASL algorithm ([Hyttinen et al.](https://arxiv.org/abs/1303.2871)) efficiently enumerates all causal-timescale graphs that could have generated an observed undersampled time series.  In practice this equivalence class can contain dozens of candidates — choosing among them typically requires domain knowledge or expensive additional constraints.

This project fine-tunes **Mistral-7B-Instruct** (or Llama-3-8B) with **LoRA/QLoRA** on synthetic training examples generated by gunfolds' own causal simulator.  No public dataset covers this task — every training example is produced from the same ground-truth causal simulator used in our ICML work, making the fine-tuning domain-specific and genuinely novel.

At inference time the LLM reads the observed undersampled graph and the RASL candidate list, then outputs the most likely ground-truth DAG — acting as a learned prior over the equivalence class.

---

## Architecture

```
gunfolds pipeline                    llm-causal-select
─────────────────────────────────    ──────────────────────────────────────────
randomDAG(N)        → G*             Training data generation
undersample(G*, u)  → G_u            (data_generation/simulator.py)
drasl([G_u])        → candidates ──► GraphFormatter → (prompt, completion) JSONL
                                     LoRA fine-tuning
                                     (finetune/train.py  ←  configs/lora_config.yaml)
                                     Inference
                                     LLMCausalSelector.select(G_u, candidates, u)
                                                         └── selected DAG ──► downstream analysis
```

## Quick Start

### 1. Install dependencies

```bash
# Install gunfolds from your local clone
pip install -e /path/to/gunfolds

# Install this package
pip install -e ".[train]"
```

### 2. Generate training data

```python
from data_generation import CausalDatasetGenerator

gen = CausalDatasetGenerator(n_nodes=5, max_urate=3, capsize=20, seed=42)

train_examples = gen.generate(n_examples=2000)
eval_examples  = gen.generate(n_examples=200)

gen.save(train_examples, "data/train.jsonl")
gen.save(eval_examples,  "data/eval.jsonl")
```

### 3. Fine-tune with LoRA

```bash
python -m finetune.train --config configs/lora_config.yaml
```

Training requires a GPU with ≥24 GB VRAM for Mistral-7B in 4-bit (QLoRA).
Reduce `per_device_train_batch_size` and increase `gradient_accumulation_steps`
proportionally if VRAM is limited.

### 4. Run inference

```python
from gunfolds.solvers.clingo_rasl import drasl
from gunfolds.utils import bfutils, graphkit
from gunfolds import conversions
from inference import LLMCausalSelector

# Build an example
gt_graph = conversions.nx2graph(graphkit.randomDAG(5, degree=3))
g_u      = bfutils.undersample(gt_graph, 2)
cands    = drasl([g_u], urate=6, capsize=20)

# Load fine-tuned selector
selector = LLMCausalSelector.from_pretrained("outputs/lora-causal-select")
selected_graph, selected_index = selector.select(g_u, cands, u=2)

print(f"Selected candidate {selected_index} of {len(cands)}")
```

### 5. Evaluate

```python
from inference import SelectionEvaluator

evaluator = SelectionEvaluator(selector)
results   = evaluator.evaluate(eval_examples)
SelectionEvaluator.print_report(results)
```

## Key Hyperparameters

| Parameter | Default | Notes |
|---|---|---|
| `r` (LoRA rank) | 16 | Higher = more capacity; 8–64 typical |
| `lora_alpha` | 32 | Scaling; good heuristic: `alpha = 2r` |
| `target_modules` | `q_proj, v_proj` | Attention projections to adapt |
| `lora_dropout` | 0.05 | Regularisation on adapter weights |
| `use_4bit` | `true` | QLoRA: 4-bit NF4 base model (saves ~75% VRAM) |
| `max_seq_length` | 1024 | Truncation length; increase for larger graphs |
| `learning_rate` | 2e-4 | Standard for LoRA fine-tuning |
